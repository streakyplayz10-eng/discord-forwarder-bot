# Discord Message Forwarder Bot

Forwards messages from **live-alerts** to **himothy-alerts** and **himothy-trades**.

## Deploy on Railway

1. Connect this repo to Railway
2. Add variable: DISCORD_TOKEN = your bot token
3. Deploy - that's it!

## Getting Your Bot Token

1. Go to https://discord.com/developers/applications
2. Select your bot > Bot tab > Reset Token
3. Turn ON Message Content Intent
4. Copy the token
